# Scientific Computing for Data Analysis - Mini-project 1

This project uses Python and NumPy to perform statistical analysis on scientific data.

## How to Run 

Follow these steps to set up the project on your local machine:

### 1. Clone the repository
git clone https://github.com/riksvapn1/Scientific-Computing-for-Data-Analysis-Mini-project-1.git
cd Scientific-Computing-for-Data-Analysis-Mini-project-1

### 2. Create a Virtual Environment
python -m venv .venv

### 3. Activate the Environment
Windows (PowerShell): .venv\Scripts\activate
Mac/Linux: source .venv/bin/activate

### 4. Install Dependencies
pip install -r requirements.txt

### 5. Run the Analysis
python task_1.py
